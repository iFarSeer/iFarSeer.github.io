使用ing的Android插件

  • Android Studio
Android

Android Studio 是基于IntelliJ IDEA的Android IDE。IntelliJ IDEA本身对Plugin支持是非常出色的，在这里我分享下目前我在Android开发中使用频率比价高的插件。

安装

如何安装插件，先定位到Andorid Studio -> Preferences -> Plugins之后，在右侧下方存在Browse repositories和Install plugin from disk两个按钮,顾名思义，分别是网络安装和本地安装。

设置快捷键

我建议根据自己的习惯设置每个插件的快捷键(只包含第三方插件)。我是根据插件对用的帮助设置快捷键，很方便我记忆，每当遇到痛点时，本能帮助我迅速定位快捷键。Andorid Studio -> Preferences -> Plugins -> Keymap,搜索定位到插件，编辑自己喜欢的快捷键。

Scroll From Source

在Coding时，经常会去定位到当前编辑文件所在的package,Android Studio 提供一个按钮可以解决这个问题，习惯快捷键的我们，十分讨论频繁的触摸触摸板或者移动鼠标。Scroll From Source高效的帮助我们解决这个问题。

Scroll From Source在github上的地址,下载最新版本采取本地安装方式。

Lifecycle Sorter

在阅读代码时，我们经常了解框架，再去深入细节技术或者业务。Lifecycle Sorter能够帮助我们把Activity和Fragment中的生命周期方法按序整理代码，再也不用担心，各个生命周期方法散落在世界大地啦。在这里也提醒自己和大家，保持良好的编码规范，在Android开发中，最基础的是好好管理下我们的生命周期方法，不要等着被发现被吐槽。

对了，Lifecycle Sorter采取网络安装方式就可以啦。

Android Wifi ADB

Android Wifi ADB 是解放USB接口的利器，我是本地安装。电脑和测试手机在同一本地网络内，先使用USB把手机连接上电脑，开启插件Connected成功后，我们的USB就解放了。

ButterKnife Zelezny

ButterKnife Zelezny是ButterKnife的注解生成器，使用起来非常简单方便。本地安装和网络安装都OK。

Lua

使用网络安装搞定。经常也在项目中使用到Lua，随便安装个Lua插件很方便，类似的插件有markdown等。
